# Terapia Online - Cesar Pesoti

Este é o repositório do site oficial para informações e agendamento de sessões de Terapia Online, desenvolvido para o Psicólogo Cesar Pesoti (CRP 06/94955).

## Sobre o Site

Uma Landing Page moderna, rápida e responsiva construída para apresentar os serviços terapêuticos, especialidades e facilitar o contato direto de pacientes.

## Tecnologias Utilizadas

- **HTML5** Semântico
- **CSS3** (Puro/Vanilla) para estilo e responsividade
- **JavaScript** (Puro/Vanilla) para animações de rolagem e interatividade do menu e FAQ

## Hospedagem

O site está hospedado e possui deploy automático configurado através do **GitHub Pages**.
